# hyper-city-lights

> Dark neon theme for [Hyper](https://hyper.is/) (formerly HyperTerm).

<!-- ![](screenshot.png) -->

## Install

`git clone` to `~/.hyper_plugins/local` folder.
Then add `hyper-city-lights` to localPlugins within `~/.hyper.js` file.
